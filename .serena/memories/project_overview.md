# Awesome Agent Skills Project Overview

## Purpose
A curated list of skills, tools, and capabilities for AI coding agents. This is an "awesome list" repository that catalogs and organizes Agent Skills from various sources including official collections and community contributions.

## Key Features
- Comprehensive catalog of Agent Skills
- Organized by categories (Document Processing, Development, Data Analysis, etc.)
- Links to official tutorials and guides
- Usage instructions for different platforms (Claude, Codex, VS Code, etc.)
- Skill creation guidelines and templates
- Community resources and FAQs

## Project Structure
- `README.md` - Main documentation (English)
- `README.zh-CN.md`, `README.zh-TW.md`, `README.ja.md`, `README.ko.md`, `README.es.md` - Translations
- `.serena/` - Serena MCP configuration

## Content Categories
- **Official Skills**: Claude, OpenAI Codex, HuggingFace
- **Community Skills**: Collections and individual skills
- **Skill Categories**:
  - Document Processing
  - Development & Code Tools
  - Data & Analysis
  - Integration & Automation
  - Collaboration & Project Management
  - Security & Systems
  - Advanced & Research
