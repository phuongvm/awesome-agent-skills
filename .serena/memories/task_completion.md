# Task Completion Workflow for Awesome Agent Skills Project

## Before Completing a Task

### 1. Skill Addition Validation
If adding new skills to the list:
- [ ] Skill actually exists and is accessible
- [ ] Description is clear and accurate
- [ ] Source link is correct
- [ ] Categorized appropriately
- [ ] Platform compatibility noted

### 2. Documentation Updates
- [ ] README.md updated with new entries
- [ ] Translations updated if applicable
- [ ] Links verified and working
- [ ] Categories remain organized

## Verification Checklist
- [ ] All links are valid
- [ ] Skills are properly categorized
- [ ] Descriptions are accurate
- [ ] No duplicate entries
- [ ] Formatting is consistent

## Final Steps
1. Review `git status` and `git diff` to verify changes
2. Test all links to ensure they work
3. Verify skill categorization is correct
4. Ask user: "Is this complete from your perspective? Any adjustments needed?"

## Important Reminders
- This is a curated list, not a skill repository
- Verify skills exist before adding
- Maintain accurate links and descriptions
- Keep categories organized
- Follow contribution guidelines
