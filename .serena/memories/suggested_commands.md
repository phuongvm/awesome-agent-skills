# Suggested Commands for Awesome Agent Skills Project

## Git Commands
```bash
git status                # Check repository status
git diff                  # Review changes
git log -n 5              # Review recent commits
```

## Windows-Specific Commands
- Use `dir` instead of `ls`
- Use `cd` for directory navigation
- Use PowerShell or Git Bash
- Paths use backslashes: `O:\workspaces\oss\awesome-agent-skills`

## File Operations
- View README: `type README.md`
- View translations: `type README.zh-CN.md`
- Browse directory: `dir`

## Documentation Maintenance
- Update skill listings in README.md
- Add new categories as needed
- Verify links are working
- Update compatibility information
